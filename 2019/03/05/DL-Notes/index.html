Linear Algebra

Basic Types

  • Scalars: A scalar is just a single number, ususlly in lower-case variable names.

  • Vectors: An array of numbers, with lower case names written in bold typeface.

    x=[x1x2⋯xn]\boldsymbol{x} = \begin{bmatrix} x_1 \\ x_2 \\ \cdots \\ x_n\end{bmatrix} x=⎣⎢⎢⎡​x1​x2​⋯xn​​⎦⎥⎥⎤​

  • Matrices: A 2-D array of numbers, usually given in uppper-case variable names wiht bold typeface, Such as M\boldsymbol{M}M.

  • Tensors: An array with more than two axes, denoted by this typeface: A\boldsymbol{\mathsf{A}}A (in latex: \boldsymbol{\mathsf{A}}).

Linear Dependence and Span

If a vector m\boldsymbol{m}m is linear dependent with a vector n\boldsymbol{n}n, then m\boldsymbol{m}m can be represented by n\boldsymbol{n}n with λ\lambdaλ multiplied with it, viz. m=λn\boldsymbol{m} = \lambda \boldsymbol{n}m=λn, where λ∈R,λ≠0\lambda\in \mathbb{R}, \lambda \neq 0λ∈R,λ​=0.

The span of a set of vectors is the set of all points obtainable by linear combination of the original vectors. For example, the span of v1\boldsymbol{v_1}v1​,v2\boldsymbol{v_2}v2​ and v3\boldsymbol{v_3}v3​ is {v1+v2+v3,v1+2v2+3v3,…,mv1+nv2+kv3}\{ \boldsymbol{v_1} + \boldsymbol{v_2} + \boldsymbol{v_3}, \boldsymbol{v_1} + 2\boldsymbol{v_2} + 3\boldsymbol{v_3}, \dots, m \boldsymbol{v_1}+ n\boldsymbol{v_2} + k\boldsymbol{v_3} \}{v1​+v2​+v3​,v1​+2v2​+3v3​,…,mv1​+nv2​+kv3​}, where mmm, nnn and kkk all can be any real numbers but zero.

Euclidean norm

When the ppp is equal to 222, the Euclidean norm, L2L^2L2 came in. It is simply the euclidean distance from the origin to the point identified by xxx. The Euclidean norm is used so frequently in machine learning that it is often denoted as ∣∣x∣∣||x||∣∣x∣∣ with subscript 222 omitted. In several machine learning applications, it is important decriminate between elements that are exactly zero and elements that are very small and close to zero. Orthogonal Matrix

An orthogonal matrix is a square matrix whose rows are mutually orthogonal and whose columns are mutually orthonormal:

(4)A⊤A=AA⊤=I\boldsymbol{A^{\top} A} = \boldsymbol{A A^{\top}} = \boldsymbol{I} \tag{4} A⊤A=AA⊤=I(4)

This implies that A−1=A⊤\boldsymbol{A^{-1}} = \boldsymbol{A^{\top}}A−1=A⊤,so orthogonal matrices are of interest because their inverse is very cheap to compute.

Pay careful attention to the definition of the orthogonal matrices. Counterintuitively, their rows are note merely orthogonal but full orthonormal.

Eigendecomposition

In this loooong eigendecomposition, we decompose a matrix into a set of eigenvectors and eigenvalues.

An eigenvectors of a square matrix A\boldsymbol{A}A is a non-zero vector v\boldsymbol{v}v such that multiplication by A\boldsymbol{A}A alters only the scale of v\boldsymbol{v}v: Av=λv\boldsymbol{Av} = \lambda \boldsymbol{v}Av=λv. The scalar λ\lambdaλ is known as eigenvalue corresponding to this eigenvector.

If v\boldsymbol{v}v is an eigenvector of A\boldsymbol{A}A, then so is any rescaled vector svs\boldsymbol{v}sv for s∈R,s≠0s \in \mathbb{R}, s \neq 0s∈R,s​=0. Moreover, svs \boldsymbol{v}sv still has the same eigenvalue. For this reason, we only look for unit eigenvectors.

Suppose a matrix A\boldsymbol{A}A has nnn linear independent eigenvectors, and we use these eigenvectors to form a matrix V\boldsymbol{V}V with one eigenvector per column. Likewise, we concatenate the eigenvalues to form a vector λ\boldsymbol{\lambda}λ. The eigendecomposition of A\boldsymbol{A}A is then given by

(5)A=Vdiag(λ)V−1\boldsymbol{A} = \boldsymbol{V} \text{diag}(\boldsymbol{\lambda}) \boldsymbol{V^{-1}} \tag{5} A=Vdiag(λ)V−1(5)

Specifically, every real symmetric matrix can be decomposed into an expression using only real-valued eigenvectors and eigenvalues:

(6)A=QΛQ−1\boldsymbol{A} = \boldsymbol{Q} \boldsymbol{\Lambda} \boldsymbol{Q^{-1}} \tag{6} A=QΛQ−1(6)

where Q\boldsymbol{Q}Q is an orthogonal matrix composed of eigenvectors of A\boldsymbol{A}A, and Λ\boldsymbol{\Lambda}Λ is a diagonal matrix. The eigenvalue Λi,j\Lambda_{i,j}Λi,j​is associated with the eigenvectors in column iii of Q\boldsymbol{Q}Q, denoted as Q:,i\boldsymbol{Q}_{:, i}Q:,i​. Because Q\boldsymbol{Q}Q is an orthogonal matrix, we can think of A\boldsymbol{A}A as scaling space by λi\lambda_iλi​ in direction v(i)\boldsymbol{v^{(i)}}v(i) .

The eigendecomposition of a matrix tells us many useful facts about the matrix.

  • The matrix is singular if and only if any of the eigenvalues are zero.
  • The eigendecomposition of a real symmetric matrix can alse be used to optimise quadratic expression of the form f(x)=x⊤Axf(\boldsymbol{x}) = \boldsymbol{x^{\top}Ax}f(x)=x⊤Ax subject to ∣∣x∣∣2=1||\boldsymbol{x}||_2 = 1∣∣x∣∣2​=1. Whenever x\boldsymbol{x}x is equal to an eigenvector of A\boldsymbol{A}A, fff takes on the value of the corresponding eigenvalue. The maximum value of fff within the constraint region is the maximum value of eigenvalue and its minimum value within the constraint region is the minimum eigenvalue.

A matrix whose eigenvalues are all positive is called positive definite. A matrix whose eigenvalues are all non-negative is called positive semidefinite. Likewise, so are the negative definite and negative semidefinite.

Positive semidefinite matrices are interesting because they guarantee that ∀x,x⊤Ax≥0\forall \boldsymbol{x}, \boldsymbol{x^{\top}Ax} \geq 0∀x,x⊤Ax≥0.

Positive definite matrices additionally guarantee that x⊤Ax=0  ⟹  x=0\boldsymbol{x^{\top}Ax} = 0 \implies \boldsymbol{x} = \boldsymbol{0}x⊤Ax=0⟹x=0

Singular Value Decomposition (SVD)

Every matrix has a singular value decomposition, but the same is not true for the eigenvalue decomposition. The singular decomposition is similar to eigendecomposition, except this time we will write A\boldsymbol{A}A as a product of three matrices:

(7)A=UDV⊤\boldsymbol{A} = \boldsymbol{UDV^{\top}} \tag{7} A=UDV⊤(7)

Suppose that A\boldsymbol{A}A is an m×nm \times nm×n matrix. Then U\boldsymbol{U}U is defined to be an m×mm \times mm×m matrix, D\boldsymbol{D}D to be an m×nm \times nm×n matrix, and V\boldsymbol{V}V to be an n×nn \times nn×n matrix. The matrix U\boldsymbol{U}U and V\boldsymbol{V}V are both defined to be orthogonal matrices. The matrix D\boldsymbol{D}D is defined to diagonal matrix.

Note that D\boldsymbol{D}D is not necessary square.

The elements along the diagonal of D\boldsymbol{D}D are singular values. The columns of U\boldsymbol{U}U and V\boldsymbol{V}V are left-singular vectors and right-singular vectors respectively.

The left-singular vectors of A\boldsymbol{A}A are the eigenvectors of AA⊤\boldsymbol{AA^{\top}}AA⊤. And the right-singular vectors of A\boldsymbol{A}A are the eigenvectors of A⊤A\boldsymbol{A^{\top}A}A⊤A.

The Moore-Penrose Pseudoinverse

The pseudoinverse of A\boldsymbol{A}A is defined as a matrix

(8)A+=lim⁡α→0(A⊤A+αI)−1A⊤\boldsymbol{A^{+}} = \displaystyle \lim_{\alpha \to 0}(\boldsymbol{A^{\top}A} + \alpha\boldsymbol{I})^{-1}\boldsymbol{A^{\top}} \tag{8} A+=α→0lim​(A⊤A+αI)−1A⊤(8)

Practical algorithms for computing the pseudoinverse are not based on this definition, but rather the formula

(9)A+=VD+U⊤\boldsymbol{A^{+}} = \boldsymbol{VD^+U^{\top}} \tag{9} A+=VD+U⊤(9)

where V\boldsymbol{V}V, D\boldsymbol{D}D and U\boldsymbol{U}U are the singular value decomposition of A\boldsymbol{A}A, and the pseudoinverse D+\boldsymbol{D^+}D+ of a diagonal matrix D\boldsymbol{D}D is obtained by taking the resiprocal of its non-zero elements then taking the transpose of the resulting matrix.

When A\boldsymbol{A}A has more columns than its rows, it provides the solution x=A+y\boldsymbol{x} = \boldsymbol{A^+y}x=A+y with minimal Euclidean norm ∣∣x∣∣2||\boldsymbol{x}||_2∣∣x∣∣2​ among all possible solutions.

When A\boldsymbol{A}A has more rows than its columns, it provides the x\boldsymbol{x}x which Ax\boldsymbol{Ax}Ax is as close as possible to y\boldsymbol{y}y in terms of Euclidean norm ∣∣Ax−y∣∣2||\boldsymbol{Ax}-\boldsymbol{y}||_2∣∣Ax−y∣∣2​

The Determinant

The determinant of a square matrix, denoted as det(A)\text{det}(\boldsymbol{A})det(A) is a function that maps the matrices to real scalars. The determinant is equal to the product of all the eigenvalues of the matrix. The absolute value of it can be thought of as a measure of **how much multiplications by the matrix expands or contracts space.

Probability And Information Theory

Three possible source of uncertainty:

  • Inherent stochasticity in the system being modeled. For example, we can crate a theoretical scenarios that we postulate to have random dynamics, such as a hypothetical card game, where we assume the cards are truly shuffled into a random order.
  • Incomplete observations. Even deterministic systems can appear stochastic when we cannot observe all variables that drive the behaviour of the system.
  • Incomlete modeling. When we use a model that must discard some of the information we have observed, the discarded information might result in uncertainty in the model’s predictions.

In many cases, it is more practical to use a simple but uncertain rule, rather than a complex but certain one.

Frequentist probability V.S. Bayesian probability

The frequentist sees the probability as the long-run expected frequency of occurrence. P(A)=n/NP(A) = n/NP(A)=n/N, where nnn is the number of times even AAA occurs in the NNN opportunities.

The Bayesian view of probability is related to degree of belief. It is a measure of the plausiblity of an event given imcomplete knowledge.

Gaussian Distribution

The most commonly used distribution over real numbers is the normal distribution, also known as Gaussian distribution:

(16)N(x;μ,σ2)=12πσ2exp(−12σ2(x−μ)2)\mathcal{N}(x;\mu, \sigma^2) = \sqrt{ \frac{1}{2\pi\sigma^2} } \text{exp} \left( - \frac{1}{2\sigma^2}(x-\mu)^2\right) \tag{16} N(x;μ,σ2)=2πσ21​​exp(−2σ21​(x−μ)2)(16)

The two parameters μ∈R\mu \in \mathbb{R}μ∈R and σ∈(0,∞)\sigma \in (0, \infty)σ∈(0,∞) control the normal distribution. The parameter μ\muμ gives the corrodinate of the central peak. This is also the mean of the distribution: E[x]=μ\mathbb{E}[\text{x}] = \muE[x]=μ. The standard deviation of the distribution is given by the σ\sigmaσ, and the variance by σ2\sigma^2σ2.

Many distributions we wish to model are truly close to being normal distributions. The central limit theorem shows that the sum of many independent random variables is approximately normally distributed.

The normal distribution encodes the maximum amount of the uncertainty over the real numbers.

The normal distribution generalizes to Rn\mathbb{R}^nRn, in which case it is knonw as the multivariate normal distribution. It may parametrized with a positive definite (with all positive λ\lambdaλ, A.K.A, eigenvalues) symmetric matrix Σ\boldsymbol{\Sigma}Σ:

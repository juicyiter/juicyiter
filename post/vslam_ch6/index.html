Given a linear equation \(\boldsymbol{Ax} = \boldsymbol{b}\), prove that its least square root is \(\boldsymbol{x} = (\boldsymbol{A}^\top \boldsymbol{A})^{-1} \boldsymbol{A}^\top \boldsymbol{b}\), when the coefficient matrix is overdetermined.

Define \(f(\boldsymbol{x}) = \boldsymbol{Ax} - \boldsymbol{b}\). The root of least square root of the linear equation is

\begin{equation} \underset{\boldsymbol{x}}{\mathrm{argmin}} \frac{1}{2} ||f(\boldsymbol{x})||^2_2 = \frac{1}{2} (\boldsymbol{x}^\top \boldsymbol{A}^\top \boldsymbol{Ax} - 2 \boldsymbol{b}^\top \boldsymbol{Ax} + \boldsymbol{b}^\top \boldsymbol{b}) \end{equation}

The Jacobian is:

\begin{equation} \boldsymbol{J}(\boldsymbol{x}) = \boldsymbol{A}^\top \boldsymbol{Ax} - \boldsymbol{A}^\top \boldsymbol{b} \end{equation}

So the root of the linear equation is equal to the root of the \(\boldsymbol{J}(\boldsymbol{x}) = \boldsymbol{0}\):

\begin{equation} \boldsymbol{x} = (\boldsymbol{A}^\top \boldsymbol{A})^{-1} \boldsymbol{A}^\top \boldsymbol{b} \end{equation}

Comparison between Gradient descent , Newton’s method, Gaussian-Newton and Levenberg-Marquadt 1

Scalar-valued goodness-of-fit measure is called chi-squared error criterion:

\begin{equation} \begin{aligned} \chi^{2}(\boldsymbol{p}) &=\sum_{i=1}^{m}\left[\frac{y\left(t_{i}\right)-\hat{y}\left(t_{i} ; \boldsymbol{p}\right)}{\sigma_{y_{i}}}\right]^{2} \\
&=(\boldsymbol{y}-\hat{\boldsymbol{y}}(\boldsymbol{p}))^{\top} \boldsymbol{W}(\boldsymbol{y}-\hat{\boldsymbol{y}}(\boldsymbol{p})) \\
&=\boldsymbol{y}^{\top} \boldsymbol{W} \boldsymbol{y}-2 \boldsymbol{y}^{\top} \boldsymbol{W} \hat{\boldsymbol{y}}+\hat{\boldsymbol{y}}^{\top} \boldsymbol{W} \hat{\boldsymbol{y}} \end{aligned} \end{equation}

Typically the weight matrix is diagonal with \(\boldsymbol{W}_{ii} = 1/\sigma_{y_i}^2\).

  • Gradient descent

    • Updates parameter values in the “downhill” direction
    • By \(\Delta{\boldsymbol{p}} = \alpha \boldsymbol{J}^{\top} \boldsymbol{W}(\boldsymbol{y}-\hat{\boldsymbol{y}})\)
    • Converges well with simple objective function.

  • Gaussian-Newton (first-order approximation) 2

    • Presumes that the objective function is quadratic in parameters near the optimal solution and finds the minimum of the quadratic.
    • Converges faster than the gradient descent for moderately-sized problems.
    • \(\frac{\partial}{\partial \Delta{\boldsymbol{p}}} \chi^{2}(\boldsymbol{p}+\Delta{\boldsymbol{p}}) \approx-2(\boldsymbol{y}-\hat{\boldsymbol{y}})^{\top} \boldsymbol{W} \boldsymbol{J}+2 \Delta{\boldsymbol{p}}^{\top} \boldsymbol{J}^{\top} \boldsymbol{W} \boldsymbol{J}\)
    • Updates by the result of the normal equation \(\left[\boldsymbol{J}^{\top} \boldsymbol{W} \boldsymbol{J}\right] \boldsymbol{h}_{\mathrm{gn}}=\boldsymbol{J}^{\top} \boldsymbol{W}(\boldsymbol{y}-\hat{\boldsymbol{y}})\)

  • Newton’s method (second-order approximation)3

  • Levenberg-Marquadt 4

    • Adaptively varies the parameter updates between the gradient descent update and the Gauss-Newton update. The Levenberg-Marquadt method acts more like a gradient-descent method when the parameters are far from their optimal value, and acts more like the Gauss-Newton method when the parameters are close to their optimal value.
    • \(\left[\boldsymbol{J}^{\top} \boldsymbol{W} \boldsymbol{J}+\lambda \boldsymbol{I}\right] \boldsymbol{h}_{\mathrm{lm}}=\boldsymbol{J}^{\top} \boldsymbol{W}(\boldsymbol{y}-\hat{\boldsymbol{y}})\)
    • Small values of damping parameter \(\lambda\) result in a Gauss-Newton update and large values of \(\lambda\) result in a gradient descent update.
    • \(\lambda\) is initiated to be large so that the first update are small steps in the steepest direction, as the solution improves, \(\lambda\) is decreased, the LM approaches GN, and the solution typically accelerates to the local minimum;
